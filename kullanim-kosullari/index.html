Kullanım Koşulları

1. Genel

Bu web sitesine erişiminiz ve kullanımınız, aşağıda belirtilen kullanım koşullarına tabidir. Web sitemizi kullanarak bu koşulları kabul etmiş sayılırsınız. Bu koşulları kabul etmiyorsanız lütfen web sitemizi kullanmayınız.

2. Fikri Mülkiyet Hakları

Bu web sitesinde yer alan tüm içerik, görseller, logolar, tasarım, metin, grafik ve diğer materyaller Frontal Alüminyum İnş. San. Tic. A.Ş.'ye aittir ve Türkiye Cumhuriyeti fikri mülkiyet mevzuatı kapsamında korunmaktadır.

Yazılı izin alınmaksızın bu materyallerin kopyalanması, çoğaltılması, dağıtılması, yeniden yayınlanması veya ticari amaçla kullanılması yasaktır.

3. Site Kullanım Kuralları

Web sitemizi kullanırken aşağıdaki kurallara uymayı kabul edersiniz:

  • Web sitesini yalnızca yasal amaçlarla kullanmak
  • Web sitesine zararlı yazılım, virüs veya benzeri kod yüklemekten kaçınmak
  • Web sitesinin güvenliğini tehlikeye atacak eylemlerden kaçınmak
  • Diğer kullanıcıların web sitesini kullanmasını engelleyecek faaliyetlerde bulunmamak

Teknik nedenlerle veya bakım çalışmaları sebebiyle web sitemize erişim geçici olarak kesintiye uğrayabilir. Bu tür kesintilerden dolayı sorumluluk kabul edilmez.

4. İçerik Doğruluğu

Web sitemizdeki bilgiler güncel ve doğru tutulmaya çalışılmakla birlikte, bilgilerin eksiksizliği veya doğruluğu konusunda herhangi bir garanti verilmemektedir. Web sitesinde yer alan proje görselleri temsili nitelikte olabilir.

Ürün ve hizmetlerimize ilişkin güncel ve detaylı bilgi almak için doğrudan bizimle iletişime geçmenizi öneririz.

5. Sorumluluk Sınırlaması

Frontal Alüminyum İnş. San. Tic. A.Ş., web sitesinin kullanımından veya kullanılamamasından doğabilecek doğrudan veya dolaylı zararlardan sorumlu tutulamaz. Web sitesindeki bilgilere dayanarak alınan kararlardan kaynaklanan sonuçlardan sorumluluk kabul edilmez.

6. Bağlantılı Siteler

Web sitemiz, üçüncü taraf web sitelerine bağlantılar içerebilir. Bu bağlantılar yalnızca bilgilendirme amacıyla sunulmaktadır. Frontal Alüminyum, bağlantı verilen web sitelerinin içeriği, gizlilik politikaları veya uygulamalarından sorumlu değildir.

7. İletişim Formu

Web sitemizdeki iletişim formu aracılığıyla gönderilen bilgilerin doğruluğu ve güncelliği, bilgileri gönderen kişinin sorumluluğundadır. Yanlış veya eksik bilgi verilmesinden kaynaklanan sorunlardan Frontal Alüminyum sorumlu tutulamaz.

8. Değişiklikler

Bu Kullanım Koşulları, önceden bildirmeksizin güncellenebilir. Yapılan değişiklikler web sitemizde yayınlandığı tarihte yürürlüğe girer. Web sitemizi kullanmaya devam etmeniz, güncellenmiş koşulları kabul ettiğiniz anlamına gelir.

9. Uygulanacak Hukuk

Bu Kullanım Koşulları, Türkiye Cumhuriyeti hukukuna tabidir. Bu koşullardan kaynaklanan veya bunlarla bağlantılı her türlü uyuşmazlıkta Ankara Mahkemeleri ve İcra Daireleri yetkilidir.

Alüminyum cephe sistemlerinde 20 yılı aşkın tecrübe ile Türkiye'nin önde gelen firması.
